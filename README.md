
# Sportsee

Coaching Sportif


## Technologies

JS, React, Sass, html , Recharts, Node, Docker



## Installation

Installation

```bash
  cd Sportsee
  git clone https://github.com/Gigeone/P12OC-SportSee.git
  npm install

```
    
Lancement du backend

```bash
  cd Sportsee
  cd .\P12-front-end-dashboard-back-end\
  npm run start
```

Lancement du frontend

```bash
  cd Sportsee
  npm run dev
```
